Fire your files to any repository as a Pull Request

## Installation

```bash
npm install -g pr-cannon
```

## Usage

```bash
# Basic usage
pr-cannon fire <file> <owner/repo>

# Specify destination path
pr-cannon fire <file> <owner/repo> --path <destination>
```

## Development

```bash
# Install dependencies
npm install

# Build
npm run build

# Development mode
npm run dev
```
